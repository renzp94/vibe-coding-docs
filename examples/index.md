# 示例

这里提供一系列 Vibe Coding 实践示例，从简单到复杂。

## 基础示例

### 1. Todo 应用
一个完整的待办事项应用，展示基础 CRUD 操作。

### 2. 天气预报
调用 API 获取天气数据并展示。

### 3. 计算器
支持基本运算的科学计算器。

## 进阶示例

### 1. 博客系统
包含文章列表、详情页、评论功能。

### 2. 实时聊天室
使用 WebSocket 的即时通讯应用。

### 3. 数据可视化仪表盘
使用图表库展示业务数据。
